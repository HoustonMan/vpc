# 删除对等连接<a name="ZH-CN_TOPIC_0075677489"></a>

## 功能介绍<a name="section9999113915104"></a>

删除对等连接。

可以在在本端或对端任何一端删除对等连接。

## URI<a name="section129990396107"></a>

DELETE /v2.0/vpc/peerings/\{peering\_id\}
