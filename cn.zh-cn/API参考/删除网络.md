# 删除网络<a name="ZH-CN_TOPIC_0201534300"></a>

## URI<a name="section29996404205035"></a>

## 请求消息<a name="section54655929205035"></a>

无。

## 响应消息<a name="section48616095205035"></a>

```

```

响应样例
